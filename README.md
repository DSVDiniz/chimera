# My Extension


## Based on `focus-editor`
- **Next Error** (`my-extension.nextError`): Jump to the next error in the workspace.
- **Align Cursors** (`my-extension.alignCursors`): Align cursors horizontally by inserting spaces.
- **Cycle Casing** (`my-extension.cycleCasing`): Cycle through casing formats (camel, pascal, snake, kebab, etc.). Order configurable via `my-extension.cycleCasingOrder`.
- **Scroll Up Fast** (`my-extension.scrollUpFast`): Scroll up by a configured number of lines (default 3). Configurable via `my-extension.scrollFastLineCount`.
- **Scroll Down Fast** (`my-extension.scrollDownFast`): Scroll down by a configured number of lines.

## Based on `sublime-text`
- **Sort Lines** (`my-extension.sortLines`): Sort selected lines (case insensitive).
- **Sort Lines (Case Sensitive)** (`my-extension.sortLinesCaseSensitive`): Sort selected lines (case sensitive).
- **Unique Lines** (`my-extension.uniqueLines`): Remove duplicate lines from selection.
- **Reverse Lines** (`my-extension.reverseLines`): Reverse the order of selected lines.
- **Shuffle Lines** (`my-extension.shuffleLines`): Randomly shuffle selected lines.

## Based on another extension that doesn't work/exist anymore
- **Add Numbers to Cursors** (`my-extension.addNumbersToCursors`): Insert sequential numbers at each cursor position.

## Argument Formatting
- **Split Arguments** (`my-extension.splitArguments`): Split comma-separated values onto separate lines (smart handling of quotes/parens).
- **Unsplit Arguments** (`my-extension.unsplitArguments`): Join multi-line comma-separated values into a single compact line.
